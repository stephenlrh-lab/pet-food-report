## 2. 上游分析框架

### 2.1 年报中对应章节

| 章节 | 位置提示 | 提取内容 |
|---|---|---|
| 主营业务分析 → 成本构成 | 年报 第三节 管理层讨论与分析 | 原材料占营业成本比重、前五大主要原材料类型 |
| 采购模式 | 主营业务分析 - 采购模式 | 采购策略、供应商管理制度、原料质量标准 |
| 主要供应商 | 前五名供应商采购情况 | 前五大供应商名称、采购金额、占比 |
| 存货 / 原材料 | 财务报表附注 → 存货 | 原材料项、库存商品项金额 |
| 关联采购 | 关联交易 | 是否有关联方原材料供应 |

### 2.3 上游分析关键指标

| 指标 | 数据来源 | 分析意义 |
|---|---|---|
| 原材料占营业成本比重 | 年报 成本分析表 | 判断原料价格波动对企业利润的敏感性 |
| 前五大供应商集中度 | 年报 主要供应商 | >50% 说明上游依赖度高、议价能力弱 |
| 进口原料占比 | 年报 / 海关数据 | 汇率风险敞口、替代国产原料的可行性 |
| 鸡肉 / 鱼粉价格走势 | 饲料工业协会 / 海关 | 核心原料成本趋势判断 |

## 3. 下游分析框架

### 3.1 年报中对应章节

| 章节 | 位置提示 | 提取内容 |
|---|---|---|
| 营业收入构成 | 主营业务分析 - 营业收入构成 | 分产品、分地区、分销售模式的收入金额和占比 |
| 销售模式 | 主营业务分析 - 销售模式 | ODM/OEM/自有品牌/直销/经销比例 |
| 主要客户 | 前五名客户销售情况 | 前五大客户名称、销售金额、占比 |
| 境内外收入 | 分地区 收入拆分 | 境内 vs 境外收入及毛利率对比 |
| 产能利用 | 产能与开工情况 | 各工厂产能、产量、产能利用率 |
| 在建工程 | 财务报表附注 → 在建工程 | 扩产方向判断 |

### 3.2 下游分析关键指标

| 指标 | 数据来源 | 分析意义 |
|---|---|---|
| 前五大客户集中度 | 年报 主要客户 | >60% 意味着大客户依赖风险 |
| 境内 / 境外收入比 | 年报 分地区收入 | 判断国际化程度和地缘政治风险 |
| OEM/ODM vs 自有品牌比 | 年报 销售模式 | 判断从代工向品牌转型的进度 |
| 毛利率 (分产品/分渠道) | 年报 毛利率 | OEM 通常 15-25%、自有品牌 35-55% |
| 产能利用率 | 年报 产能 | >85% 意味着有扩产需求 |

## 4. 三家公司上下游预判 (需年报核实)

| 维度 | 佩蒂股份 | 中宠股份 | 乖宝宠物 |
|---|---|---|---|
| **上游侧重** | 咬胶用牛皮/鸡肉、肉质零食原料 | 鸡肉、鱼、罐头原料 | 主粮用鸡肉粉/谷物、功能添加剂 |
| **海外采购依赖** | 新西兰/澳洲羊肉鹿肉 (高端线) | 美国子公司本地采购 | 较低 (国产替代强) |
| **下游模式** | OEM/ODM 为绝对主力 | OEM + 自有品牌 (Wanpy) 双轮 | 自有品牌 (麦富迪) 占比高 |
| **海外收入占比** | 80%+ | 70%+ | 较低 (以国内市场为主) |
| **客户集中度** | 前五大客户占比可能较高 | 中等 | 分散 (C 端为主) |
| **毛利率** | OEM 主导，偏低 | 代工+品牌，中等 | 自有品牌主导，偏高 |

> 上表为预判值，实际数据需从年报中核实并更新。
